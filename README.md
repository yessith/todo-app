# Todo App

Aplicativo web para crear una lista de tareas "Todo" para mantener un orden diario de las diversas labores del dia a dia, los usuarios pueden crear una nueva tarea, editarla y una vez completada pueden eliminar dicha tarea de la lista, de igual forma tienen la posibilidad de filtrar las tareas que ya se encuentran completadas o sin completar.
